## What I Build

I architect and ship large-scale software systems for real businesses. Here's what's live in production:

### UPVC Cloud ERP — Enterprise Resource Planning for Pakistan's Construction Industry
- Full-scale ERP system built from scratch serving **15+ companies** managing **billions of PKR** in annual revenue
- Modules: Inventory Management, Financial Accounting (double-entry bookkeeping), CRM, Project Tracking, Custom Canvas-based Window Designer
- Stack: **Angular + Node.js + PostgreSQL + Docker + AWS (EC2, RDS, S3, CloudFront)**
- Features: Offline-first with service workers, real-time sync, audit logging, multi-tenant architecture
- Impact: 30-40% reduction in operational overhead, 80% faster invoice processing, zero data loss incidents

### PowerSell — Gamified Real Estate Platform for the US Market
- Built MVP in **35 days** with a team of 6 developers — launched January 2026
- Full deal pipeline (Acquisitions, Dispositions, Closings) with real-time WebSocket updates
- Gamification engine: $10K 30-Day Challenge, leaderboards, spin wheel rewards, referral system
- Multi-portal architecture: Closer, Admin, Setter, Main Platform + Influencer Portal
- Stack: **TypeScript, WebSockets, WebHooks, Role-based Access Control**
- Visit: [powersell.io](https://powersell.io)

### REIVEX Technologies — Custom Enterprise Software for Emerging Markets
- My company focused on building affordable, domain-specific enterprise software for South Asian markets
- Philosophy: Silicon Valley quality at emerging market prices
